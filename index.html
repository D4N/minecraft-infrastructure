Minecraft Server As a Service¶

Ziel ist es eine Server Struktur aufzubauen bei der die Betriebskosten möglichst gering sind, die Verfügbarkeit und die usability sollte allerdings mit einem Anblieter wie nitrado.com vergleichbar sein.

services
  • Automatisches Backup
  • Prüfen der Welten mit region fixer
  • Scriptgesteuerte vollständiege Bereitstellung

Genutzte Projekte¶

Für die unterschiedlichen Services werden weitere OpenSource Projekte verwendet.

Projekte description
minecraft-plugin-archivator Minecraft Java Plugin
ansible-minecraft Ansible Role to Controll the Server
ansible-minecraft-region-fixer Ansible Role für den region Fixer
itzg/rcon-cli Go Rcon Commandline Interface
ansiblebit/oracle-java Ansible Role for install Oracle Java
geerlingguy/ansible-role-certbot Ansible Role for LetsEncrypt CertBot
geerlingguy/ansible-role-nginx Ansible Role for handle the nginx
