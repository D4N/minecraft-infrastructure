Neuer Spieler¶

Neue Spieler müssen im Playbook Verzeichniss unter vars/whitelist_players.yml in die Whitelist whitelist_users eingetragen werden.
Neben der Whitelist müssen auch noch die PermissionsEx gesetzt werden.
Ohne das explizite setzen kann der spieler nicht viel machen außer rumrennen ;)
/pex user <user> group add <group>

Groups¶

gamers_admins
Gruppe für alle Admins, User dieser gruppe dürfen alles (-'*')!
gamers_creative_builder
Gruppe für Spieler auf der Creative World, erlaubt z.B. die Nutzung von WorldEdit.
gamers_survival_builder
Gruppe für Spieler auf der Survival Map, erlaubt klassische Spieler Interaktionen, wie z.B. Klötzchen stapeln.
